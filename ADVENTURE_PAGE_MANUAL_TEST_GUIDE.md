# 모험 페이지 수동 테스트 가이드

## 테스트 준비
1. 개발 서버 실행: `npm run dev`
2. 브라우저에서 http://localhost:3000/adventure 접속
3. 스크린샷 저장 위치: `C:\Users\USER\Pictures\Screenshots\`
